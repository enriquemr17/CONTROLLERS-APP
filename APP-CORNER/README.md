# APP CORNER

Aplicación web desarrollada con React, TypeScript y Vite.

## Tecnologías

- React + TypeScript
- Vite
- Tailwind CSS
- React Router

## Instalación

```bash
npm install
npm run dev
```

## Estructura del proyecto

src/
├── api/
├── components/
├── context/
├── hooks/
├── pages/
├── types/
└── utils/
server/
└── src/
├── config/
├── controllers/
├── routes/
└── services/
