# Despliegue

## Frontend (Vercel)
- Conectar repositorio de GitHub en vercel.com
- Root Directory: APP-CORNER
- Build Command: npm run build
- URL: https://controllers-app.vercel.app

## Backend (Render)
- Conectar repositorio de GitHub en render.com
- Root Directory: APP-CORNER/server
- Build Command: npm install
- Start Command: node src/index.js
- URL: https://controllers-app.onrender.com
