# Retrospectiva del proyecto

## Qué aprendí
He aprendido a utilizar TypeScript y a conectar un proyecto a una base de datos. 
También he conseguido mayor fluidez a lo largo del proyecto. 

## Cómo conecté frontend, backend y API
El frontend en React consume la API de RAWG para buscar juegos y se comunica 
con el backend Express mediante fetch. El backend gestiona la colección del 
usuario y expone endpoints REST que el frontend consume desde src/api/.

## Principales problemas encontrados
El despliegue de vercel me ha dado bastantes probelmas hasta conseguir que funcionase. 

Al ser un lenguaje que no había utilizado antes, ha sido un poco más complicado el aprendizaje; Lo más costoso fue la creación de context. 

## Cómo utilicé la IA
La utilicé sobre todo al principio del proyecto para saber cual debía ser el orden que tenia que llevar y para los tipados que no sabia hacer. Su uso fue disminuyendo a lo largo del proyecto al ya tener un mayor control sobre el mismo. 

## Reflexión final
Creo que es un proyecto a medio plazo, del cual podré sacar provecho más adelante y del cual puedo aprender bastante. Debo seguir con él ya que hay que pulir varias cosas, pero es un proyecto completo con una buena base.
