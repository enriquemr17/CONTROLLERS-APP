# Organización del proyecto

## Metodología
Trabajo con sprints semanales usando Trello como tablero Kanban.

## Tablero
[Ver tablero en Trello](https://trello.com/b/1Aw0oJNY/controllers-desarrollo)

## Columnas del tablero
- **Backlog**: tareas pendientes de planificar
- **In Progress**: en desarrollo activo
- **Review**: pendiente de revisión
- **Done**: completadas

## Convenciones de commits
Seguimos Conventional Commits:
- `feat:` nueva funcionalidad
- `fix:` corrección de bug
- `chore:` tareas de mantenimiento
- `docs:` documentación

## Ramas
- `main`: producción
- `dev`: integración
- `feat/nombre`: nuevas funcionalidades
